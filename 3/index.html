Cara membuat Konten/Postingan di Web wordpress

  1. Menambah konten pada pos tambah baru
  2. Yang diketik judul apa yang ingin dibuat
  3. Membuat isi dalam konten berupa paragraph
  4. Menambah gambar dengan cara klik + dan cari gambar di pencarian, kemudian unggah gambar yang ada di folder
  5. Menambah Vedio dengan cara klik + dan cari vedio dipencarian, kemudaian unggah vedio yang ada di folder
  6. menambah konten youtube di konten web, buka youtube, klik share di youtube yang ingin diambil, kemudian di embed link youtube, ketik html di pencarian konten web paste link embed tersebut kedalam html tersebut
  7. Setelah selesai membuat konten, klik terbitkan
  8. Konten akan terlihat di branda web yang dibuka
  9. Membuat konten baru seperti di penjelasan di atas
  10. Kemudian klik simpan konsep
  11. Konsep itu akan muncul berupa draf yang ada di pos
  12. Konsep digunakan ketika kita belum selesai membuatnya
  13. Konsep yang sudah di draf lalu klik sunting
  14. Kemudian belum di terbitkan Klik setting pada konten yang di sunting
  15. Kemudian di pos, kelik visibilitas berupa(public,privat,terlindung sandi), tambah sandi yang ingin dibuat
  16. Kemudian Membuat kategori baru untuk konten
  17. Kategori untuk mengkelompokan konten, Lalu terbitkan
  18. Branda konten yang dilihat akan menggunakan kata sandi karena terlindungi
  19. Pending review digunakan ketika user contributor yang membuatnya
  20. Membuat konten Scheduled yang berupa tanggal berapa yang ingin diterbitkan
  21. Cara membuat konten scheduled
  22. Pos tambah baru seperti penjelasan di atas
  23. Setelah konten dibuat lalu klik pengaturan, terus atur tanggal berapa yang ingin diterbitkan
  24. Kalau membuat tag/label, tinggal ke pengaturan, kemudian cari tag isi tag dengan nama yang ingin dibuat
  25. Cara mengaktifkan komentar pada konten
  26. pengaturan pilih komentar, kemudian klik centang izinkan berkomentar

Membuat Media di halaman Web

  1. Media tambah baru di halam web
  2. Upload file yang besar ditarik kemudian dilepas di media
  3. rik biar bisa mengupload file besar
  4. Open htacces, copy script yang di ambil di google
  5. Kemudian di paste di htacess
  6. Setelah selesai, kemudian buka pos tambah baru
  7. Membuat seperti konten dipenjelasan di atas
  8. Cara menggunakan file yang telah diupload di media
  9. Klik + baru ketik vedio, ketika sudah klik pustaka media
  10. Kemudian pilih file yang sudah di upload di media
  11. Dan hasilnya akan muncul dikonten yang dibuat
  12. Kemudian terbitkan, dan akan muncul di branda web
